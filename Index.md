## 🔄 High-Level OIDC Flow

1. User clicks **Authorize**  
2. Okta login page appears  
3. User authenticates  
4. Okta evaluates Authentication Policies  
5. Authorization Code returned  
6. Token endpoint exchanges code for tokens  
7. Application receives ID Token + Access Token
